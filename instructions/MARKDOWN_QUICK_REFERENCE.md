# Jira Markdown - Быстрая справка

## Форматирование текста

```
*жирный*          → **жирный**
_курсив_          → *курсив*
-зачеркнутый-     → ~~зачеркнутый~~
+подчеркнутый+    → <u>подчеркнутый</u>
{{код}}           → `код`
```

## Заголовки

```
h1. Самый большой заголовок
h2. Заголовок второго уровня
h3. Заголовок третьего уровня
h4. Обычный заголовок
h5. Маленький заголовок
```

## Списки

**Маркированный:**
```
* Элемент первого уровня
** Элемент второго уровня
*** Элемент третьего уровня
```

**Нумерованный:**
```
# Элемент первого уровня
## Элемент второго уровня
### Элемент третьего уровня
```

## Специальные блоки

**Цитата:**
```
bq. Текст цитаты
```

**Блок кода:**
```
{code}
const hello = "world";
console.log(hello);
{code}
```

**С указанием языка:**
```
{code:javascript}
const hello = "world";
{code}
```

**Горизонтальная линия:**
```
----
```

## Комбинирование

```
*_жирный курсив_*
-+зачеркнутый подчеркнутый+-
*{{жирный код}}*
```

## Как использовать в редакторе

### Режим "Текст" (Markdown)
1. Нажми кнопку "Текст" в правом конце тулбара
2. Редактируй текст в markdown формате
3. Выделяй текст и используй кнопки тулбара для форматирования
4. Или пиши символы вручную

### Режим "Визуально" (WYSIWYG)
1. Нажми кнопку "Визуально" 
2. Редактируй с живым превью
3. Используй кнопки тулбара для форматирования

## Синхронизация

При переключении между режимами:
- Визуальный → Текст: `**жирный**` преобразуется в `*жирный*`
- Текст → Визуальный: `*жирный*` преобразуется в `**жирный**`
- Все остальное сохраняется автоматически

## Советы

✅ **Делай так:**
- Выдели текст и используй кнопки форматирования
- Повторный клик отменяет форматирование (toggle)
- Комбинируй разные форматирования одновременно

❌ **Не делай так:**
- Не смешивай синтаксис Jira с markdown (`*текст*` работает в Jira, но `**текст**` нет)
- В режиме Текст используй только Jira синтаксис
