# Задача: Переход на Tiptap редактор (реалистичный план)

## Цель

Заменить Quill на **Tiptap** с минимальной функциональностью для демо: базовое форматирование, модульная панель, экспорт HTML/текста. Упоминания и полная локализация — опционально после базовой версии.

## Реальное состояние проекта

- **✓** Vue 3 + TypeScript уже настроены
- **✓** Базовая структура существует (RichTextEditor.vue, App.vue)
- **→** Нужна миграция `Quill` → `Tiptap`
- **→** Текущая панель инструментов — будет переработана в модульную

## MVP: Что обязательно (этап 1)

### 1. Установка зависимостей

```json
{
  "@tiptap/vue-3": "^2.x",
  "@tiptap/starter-kit": "^2.x",
  "tippy.js": "^6.x"
}
```

### 2. Компонент `TiptapEditor.vue`

Заменить `RichTextEditor.vue` с:
- `useEditor` hook для инициализации
- `EditorContent` компонент для отрисовки
- Props: `modelValue`, `toolbarItems: string[]` (какие кнопки включить)
- Events: `update:modelValue`
- Methods (expose): `getHTML()`, `getText()`, `getJSON()`, `setContent(html)`

```typescript
// Поддерживаемые команды (по умолчанию все):
const DEFAULT_TOOLBAR = [
  'bold', 'italic', 'strike',
  'heading1', 'heading2',
  'bulletList', 'orderedList', 
  'blockquote', 'codeBlock',
  'horizontalRule', 'undo', 'redo'
]
```

### 3. Компонент `ToolbarButton.vue`

Простая кнопка с:
- Props: `editor`, `command`, `isActive`, `icon`, `title`
- `isActive` → подсветка кнопки
- Click → выполнить команду редактора
- Keyboard shortcut support (Ctrl+B, Ctrl+I, и т.д.)

### 4. Обновить `App.vue`

- Убрать переключатель языка (если не критично)
- Добавить демо-контроли:
  - Переключение набора кнопок (`toolbarItems`)
  - Кнопка "Получить HTML" → показать в textarea
  - Кнопка "Очистить" → сбросить редактор
  - Resize handle оставить (как сейчас)

## Реалистичный график

| Этап | Задача | Время |
|------|--------|-------|
| 1 | Установить зависимости, создать TiptapEditor.vue | ~1–2 ч |
| 2 | Создать ToolbarButton.vue, интегрировать в редактор | ~1–2 ч |
| 3 | Обновить App.vue, протестировать базовый функционал | ~1 ч |
| 4 | Стилизация, resize handle, polish | ~1–2 ч |
| 5 | (опционально) Упоминания | ~2–3 ч |
| 6 | (опционально) Полная локализация | ~1 ч |

**Итого для MVP: 4–7 часов**

## Критерии успешного MVP

1. ✓ Проект собирается и запускается
2. ✓ Редактор отображается, принимает ввод
3. ✓ Все кнопки из `toolbarItems` работают
4. ✓ `getHTML()` и `getText()` возвращают корректные данные
5. ✓ Панель инструментов модульная (можно включать/отключать кнопки через проп)
6. ✓ Resize handle сохранен
7. ✓ Нет критических ошибок в консоли

## Что отличается от исходного плана

| Было | Стало |
|-----|-------|
| 9 шагов с полной локализацией | 4 шага базовой функциональности + опции |
| Обязательные упоминания | Упоминания отложены (опционально) |
| Полная en/ru локализация | Минимальная (только ключевые пропсы) |
| Сложная архитектура с `tippy.js` | Простой dropdown для @mentions (позже) |

## Зачем это реалистичнее

- **Не переусложняет**: демо не требует всех фич
- **Быстро**: MVP за 4–7 часов, а не за несколько дней
- **Масштабируемо**: легко добавить упоминания/локализацию потом
- **Соответствует текущему коду**: просто замена Quill на Tiptap
